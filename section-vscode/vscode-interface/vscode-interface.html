Screen zoom

Contents

Vscode 화면 비율을 확대 축소 복귀 하는 기능이 있다. 이 기능은 editor group 뿐 아니라 메뉴 사이드바 등 모든 것이 같이 확대, 축소 된다. 포토샵이나 일러스트레이터와 단축키는 같지만 vscode는 interface의 모든 부분이 확대, 축소된다.

  • Ctrl + + 또는 Ctrl + = : 확대
  • Ctrl + - : 축소
  • Ctrl + 0 : 원래크기(Zoom Level 0)
