## Explorer side bar

### Title bar
Explorer sidebar는 아래의 3가지로 구분되며, title bar를 클릭하면 해당 내용이 접혔다 펼쳐졌다 toggle된다. 내용이 많아서 스크롤이 크게 생길 경우 toggle하여 필요한 부분만 보면 된다.
- open editor
	- Vertical / Horizontal editor group layout toggle
	- Save all
	- Close all
- current working directory
	- New file
	- New folder
	- Refresh
	- Collapse All
- outline
