좋은 코드 에디터의 보편적인 조건

  • ‘무료’일 것.
  • 지속적인 업데이트와 지원(Updates & Supports)이 될 것.

    vscode는 Open source의 장점인 내부 직원이 아닌 외부 참가자들도 개발에 참여하므로
    개발에 많은 아이디어가 제공됨.(1개월에 2번 정도의 업데이트)

  • 좋은 확장성(extensible)을 가질 것.

    좋은 code editor라면 기본적으로 강력한 문서편집 기능은 당연하고, html관련 언어(css scss js ...)외의
    언어(php, asp, json, python, vue.js, react.js...) framwork 까지도 확장할 수 있는
    extension(extension은 포토샵을 예로 들면, plugin이나 add-on 같은)을 지원할 것.

  • 통합성(Integrity : All in one)

    에디터 내에서 Git, Search, Debug, CLI 등을 통합처리할 수 있는 기능을 지원해야 함.

  • 실행 속도.

    * vscode는 다른 코드에디터에 비해 실행속도가 빠르다.

    * electron으로 만들었다고 함.
