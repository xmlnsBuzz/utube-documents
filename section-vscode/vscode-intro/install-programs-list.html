필수 설치 프로그램 3개

  • Visual Studio Code (VSCode)

    SIA 채널의 코드를 실습 할 수 있는 코드에디터

    이 채널에서는 html, css, JavaScript 등 웹언어 coding은 물론
    내가 VSCode를 초창기 부터 사용해 온 VSCode에 대한 경험을 통해
    VSCode의 기본 기능 부터 고급 기능까지 설명할 것임.
    Settings, ShortCuts, Extensions

    *** 하지만 이미 다른 에디터에 익숙한 사람은 설치하지 않아도 상관없다.

  • git-scm (Source Code Management)

    작업한 code를 작업자가 필요하다 판단할 때 특정 시점(SnapShot)마다
    version을 보관하고 관리해 주는 프로그램(Linux 개발자 'Linus Torvalds'가 만듬.)

    이 채널에서는 주로 'Bash' terminal로 CLI(Command Line Interface)를 사용 할 것임.

  • Node.js

    네트워크 응용프로그램(특히 서버용) 개발 플랫폼
    이 채널에서는 bootstrap, jQuery 등 웹언어 관련 package 관리 목적으로 사용함.
