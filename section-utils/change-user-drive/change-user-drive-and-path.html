Change Windows User Account's Drive & Path
(윈도우 사용자 계정 드라이브 및 경로 변경)

  • 목적

    일반적으로 하드디스크 'C-Drive'의 '사용자(영문 OS일 경우 Users) 폴더' 계정을 사용할 경우 OS를 새로 설치하거나, 백업을 복원할 경우 사용자 계정의 데이터가 손실되는 것을 피하기 위하여 boot drive가 아닌 하드디스크로 사용자 계정 저장소를 옮긴다.

  • 방법
    1. Administrator 계정 활성화
    2. 사용자 계정에서 Log off
    3. Administrator 계정으로 로그인
    4. "C:\사용자(영문OS에서는 'Users')\사용자 계정 경로" 를 원하는 하드디스크로 복사
    5. regedit(registry editor)를 사용하여 Profilelist의 사용자 계정 경로 변경
    6. Log off 하고 사용자 계정으로 로그인
    7. 바탕화면에 .txt 파일을 만들고 드라이브 변경 여부를 확인.
  • OS 보호를 위해 Administrator 계정을 '사용안함'으로 원상복구 시킨다.
